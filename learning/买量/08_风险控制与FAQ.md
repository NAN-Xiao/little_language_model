# 第五部分：风险控制与常见问题

## 11. 风险控制

### 11.1 账号安全

```
1. 多账户策略
   - 准备 3~5 个广告账户
   - 分散预算，避免单点故障
   - 如果一个账户被封，其他账户还能跑

2. 合规要点
   - 素材必须展示真实游戏画面
   - 不要虚假宣传（"100%提现"等）
   - 避免敏感内容（暴力、色情、政治）
   - 遵守平台政策（定期查看政策更新）

3. 账户被封怎么办
   - Meta：提交申诉，通常 1~3 天有结果
   - Google：检查违规原因，整改后申诉
   - TikTok：联系代理或客服
   - 同时用备用账户继续跑
```

### 11.2 现金流管理

```
1. 设置止损线
   - Day7 ROAS < 20%：预警
   - Day7 ROAS < 10%：暂停 Campaign
   - CPI > 目标 × 2：检查原因

2. 预算控制
   - Campaign 层级设每日预算上限
   - Account 层级设总预算上限
   - 每周 review 总花费

3. 现金流预测
   - 根据 LTV 曲线预测未来收入
   - 确保账上现金能支撑到回本日
   - 预留 20~30% 缓冲
```

### 11.3 反作弊

```
常见作弊类型：
  - 假安装（机器人安装）
  - 激励安装（用户为了奖励而安装，非真实用户）
  - 点击劫持（恶意应用劫持点击）
  - 安装农场（批量刷安装）

防范措施：
  1. 使用 Adjust/AppsFlyer 的防作弊功能
  2. 监控异常数据：
     - 短时间大量安装
     - CTR 异常高（>10%）
     - 安装后立刻卸载
     - 留存率异常低
  3. 设置归因窗口：
     - 点击归因：7天（不要太长）
     - 展示归因：1天（或关闭）
  4. 排除已知作弊源
  5. 定期检查设备指纹
```

## 12. 常见问题 FAQ

**Q: 新游戏应该投多少钱测试？**

```
A: 建议准备 $5,000~$10,000 测试预算
  - Meta: $3,000~$5,000
  - Google: $1,000~$2,000
  - TikTok: $1,000~$2,000
  - ASA: $500~$1,000

测试周期：2~4周
目标：找到 CPI < LTV 的投放组合
```

**Q: CPI 一直降不下来怎么办？**

```
A: 检查清单：
  1. 素材是否过时？→ 制作新素材
  2. 定向是否太窄？→ 扩大受众
  3. 是否遇到大推期？→ 等竞品推完
  4. 游戏本身是否有吸引力？→ 优化产品
  5. 是否投了错误地区？→ 换地区测试
  6. 出价是否太低？→ 适当提高出价获得展示
```

**Q: 买量团队需要几个人？**

```
A: 不同阶段：

测试期（月预算 <$50,000）：
  - 1人（买量专员，兼素材策划）
  - 或外包给代理

成长期（月预算 $50,000~$200,000）：
  - 1个买量经理
  - 1个素材设计师
  - 1个数据分析师（可兼职）

放量期（月预算 >$200,000）：
  - 1个买量总监
  - 2~3个买量优化师（各负责1~2个渠道）
  - 1~2个素材设计师
  - 1个数据分析师
  - 1个创意策划
```

**Q: 买量代理怎么选？**

```
A: 选择标准：
  1. 是否有游戏行业经验
  2. 是否有成功案例（同类型游戏）
  3. 返点比例（通常 5~15%）
  4. 素材制作能力
  5. 数据透明度（是否共享原始数据）
  6. 服务响应速度

推荐代理（中国出海）：
  - 飞书深诺
  - 钛动科技
  - 点摩
  - Mobvista
```
