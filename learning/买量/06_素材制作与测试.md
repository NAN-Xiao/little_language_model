# 第三部分：素材制作与测试

## 9. 素材类型与制作

### 9.1 素材类型对比

| 类型 | 制作成本 | 效果 | 适合渠道 | 适合游戏 |
|------|---------|------|---------|---------|
| **玩法录屏** | 低 | 中 | 所有 | 所有 |
| **真人剧情** | 中 | 高 | TikTok, Meta | SLG、中重度 |
| **UE4/Unity 渲染** | 高 | 高 | Meta, Google | SLG、中重度 |
| **可玩广告** | 高 | 很高 | Unity, ironSource | 所有 |
| **图文卡片** | 低 | 低 | Google Display | SLG、休闲 |

### 9.2 玩法录屏素材

**制作步骤：**

```
1. 准备录制环境
   - 使用真机录制（不要用模拟器，画质差）
   - iOS: 自带录屏功能
   - Android: AZ Screen Recorder 或内置录屏
   - 分辨率：1080×1920（竖屏）

2. 录制内容
   - 展示核心玩法（前3秒必须出现）
   - 展示爽点（连击、过关、奖励）
   - 展示画面品质
   - 时长：15~30秒

3. 后期剪辑
   - 加快节奏（1.2~1.5倍速）
   - 添加字幕
   - 添加音效和背景音乐
   - 添加手指点击指示（让用户知道怎么玩）
   - 结尾加 CTA（"Download Now"）

4. 导出
   - 格式：MP4
   - 码率：10~20 Mbps
   - 文件大小：< 100MB（TikTok < 500MB）
```

### 9.3 真人剧情素材（SLG 重点）

SLG 的真人素材和休闲游戏完全不同，更注重"剧情冲突"和"情感代入"。

```
形式1：复仇/逆袭剧情
  - 开局被欺负（被其他玩家攻打、资源被抢）
  - 经过努力（升级城堡、招募英雄、结盟）
  - 最终反杀复仇
  - 结尾 CTA："你能做到吗？下载试试"
  - 适合 TikTok 和 Meta Reels

形式2：联盟战争剧情
  - 展示大规模联盟战
  -  hundreds of troops marching
  - 联盟成员语音指挥（真实感）
  - 适合 Meta 和 YouTube

形式3：建造展示
  - 从一个小村庄到宏伟帝国的时间 lapse
  - 展示建筑升级过程
  - 配合史诗音乐
  - 适合所有渠道

形式4：KOL/主播推荐
  - 找游戏主播试玩
  - 展示真实游戏体验
  - 主播解说游戏策略
  - 适合 YouTube 和 TikTok

形式5：UE4 渲染 CG
  - 高品质动画展示战争场面
  - 英雄技能特效
  - 史诗级背景音乐
  - 适合 Meta 和 YouTube
  - 制作成本高但效果好

SLG 制作要点：
  - 前3秒必须有冲突（"我的城堡被攻打了！"）
  - 展示"成长感"（从弱到强）
  - 展示"社交感"（联盟、 teamwork）
  - 加字幕（80%静音观看）
  - 使用史诗/紧张的音乐（SLG 不需要热门流行音乐）
  - 不要展示消消乐式的轻快画面（SLG 用户不吃这套）
```

### 9.4 素材测试流程

```
阶段1：内部筛选
  - 制作 10~20 个素材变体
  - 团队内部投票选出 Top 5

阶段2：小规模测试
  - 在 Meta 创建 Campaign
  - 5 个素材，每个分配 $50~$100 预算
  - 跑 24~48 小时
  - 看 IPM 和 CPI

阶段3：筛选优胜素材
  - 保留 IPM > 20 的素材
  - 淘汰 IPM < 10 的素材

阶段4：迭代优化
  - 分析优胜素材的共同特征
  - 基于共同特征制作新变体
  - 例：如果"失败瞬间"效果好，做更多"失败+你能更好"的素材

阶段5：放量
  - 把优胜素材加到主力 Campaign
  - 逐步增加预算
```

### 9.5 A/B 测试框架

```
测试原则：每次只变一个变量

测试1：素材类型
  - A: 玩法录屏
  - B: 真人剧情（复仇/逆袭）
  - C: UE4渲染
  → 看哪个 IPM 最高

测试2：前3秒钩子
  - A: "你能过这关吗？"
  - B: "99%的人过不去"
  - C: "我花了3小时才过关"
  → 看哪个 CTR 最高

测试3：CTA
  - A: "Download Now"
  - B: "Play Free"
  - C: "Try Now"
  → 看哪个 IR 最高

测试4：受众
  - A: 25-34岁女性
  - B: 35-44岁女性
  - C: 25-34岁男性
  → 看哪个 CPI 最低，LTV 最高

样本量要求：
  - 每个组至少 100 个安装才能得出结论
  - 测试周期：3~7天
```
