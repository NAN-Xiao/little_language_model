### SLG 11. 真实案例：Whiteout Survival vs Last War

这两款游戏代表了 SLG 出海的两种截然不同的命运：**一个持续增长，一个买量崩盘**。

**核心数据对比：**

| 指标 | **Whiteout Survival** | **Last War** | **一般 SLG** |
|------|----------------------|-------------|-------------|
| **上线时间** | 2022.12 | 2023.8 | — |
| **CPI（美国）** | **~$3-5**（题材破圈） | **~$8-15**（副玩法买量） | $15-30 |
| **Day1 留存** | **~42%** | **~34%** | 25-30% |
| **Day7 留存** | **~17%** | **~11%** | 8-12% |
| **Day30 留存** | **~8%** | **~4%** | 3-5% |
| **回本周期** | **~60-90 天** | **~120 天+** | 90-180 天 |
| **巅峰月投流** | 数千万美元级 | **2-3 亿美元** | $50-500万 |
| **2025 年收入** | **~21 亿美元**（全球第 4） | 从峰值回落 | — |
| **当前状态** | 持续增长 | **2026.4 全球停投** | — |

**为什么 Whiteout Survival 能赢：**

```
1. CPI 极低（$3-5 vs 行业 $15-30）
   - "冰雪末日生存"题材天然吸睛
   - 素材展示融雪、建城、求生（轻度化）
   - 吸引了大量非 SLG 用户
   - 同样的预算，买到 3-5 倍的用户量

2. 留存极高（Day1 42%，Day30 8%）
   - SLG + 模拟经营（融雪建城）
   - 前期像休闲游戏（门槛低）
   - 后期逐步引入 SLG 竞争（深度够）
   - Day30 留存是一般 SLG 的 2-3 倍

3. 回本周期短（60-90 天 vs 一般 180 天）
   - CPI $4，Day30 ARPU $2
   - Day30 ROAS ≈ 60%
   - Day60 就能回本
```

**为什么 Last War 崩盘了：**

```
1. 留存是致命伤
   - 素材是跑酷、射击副玩法
   - 用户被"骗"进来，打开发现是 SLG
   - Day30 留存只有 4%（Whiteout 的 50%）

2. 买量成本持续攀升
   - 2024 年初 CPI ~$8
   - 2025 年 CPI ~$15+
   - 回本周期从 90 天拉长到 120 天+

3. 2026 年 4 月全球停投
   - 雪球披露：后期买一个用户回本周期超过 120 天
   - 越买越亏，被迫止损
   - 游戏还在运营，但不再获取新用户
```

**两款游戏的买量策略对比：**

| 维度 | Whiteout Survival | Last War |
|------|------------------|----------|
| **素材类型** | 融雪建城、生存挑战 | 跑酷、射击副玩法 |
| **素材与游戏一致性** | 一致 | 割裂（误导） |
| **CPI 优势来源** | 题材破圈 | 副玩法低门槛 |
| **留存优势来源** | 玩法融合 | 无（素材骗进来就流失） |
| **买量持续性** | 持续 2 年+ | 1.5 年后崩盘 |
| **收入天花板** | 21 亿美元/年 | 峰值后回落 |

**核心启示：**

```
买量不是比谁花的钱多，而是比谁的用户"来得多、留得住、花得久"。

Last War 巅峰月投 2-3 亿美元（是 Whiteout 的数倍）
但最终因为留存崩盘而停投

Whiteout 靠低 CPI 和高留存实现了
"花更少的钱，赚更多的钱"
```

**Last War 崩盘的数学分析：**

用修正公式 $N = \frac{T \times R}{R-1-c}$ 具体计算：

```
假设 Last War 巅峰期：
  - 月买量：$200,000,000
  - 月运营：$80,000,000（c = 0.4）
```

**阶段1：2024 年初（CPI ~$8，黄金期）**

```
CPI = $8，T = 3 个月（90 天回本）
估算 LTV = $24，R = 24/8 = 3.0

      3 × 3     9
  N = ───── = ─── = 7.5 个月
      3-1-0.4   1.6

垫资峰值（简化模型）：
  Month 1-3：支出 $840M，收入 ≈ $100M
  Month 4-6：支出 $840M，收入 ≈ $400M
  Month 7：当月打平

  垫资峰值 ≈ $700-900M（7-9 亿美元）
  （旧版错误公式算出来 10 个月，偏保守）
```

**阶段2：2025 年（CPI ~$15，崩盘期）**

```
CPI = $15，T = 4 个月（120 天回本）
LTV 假设不变 = $24，R = 24/15 = 1.6

检查前提：R = 1.6 < 1+c = 1.4？
  1.6 > 1.4，理论上能回本
  但 N = 4 × 1.6 / (1.6-1-0.4) = 6.4 / 0.2 = 32 个月
  → 回正时间太长，CPI 还在涨，实际回不了本 ❌

这就是停投的数学原因：
  CPI 涨了 87%，LTV 没变
  单位经济模型从"盈利"变"破产"
```

**CPI 上涨的连锁反应：**

| 指标 | 2024 年初 | 2025 年 | 变化 |
|------|----------|---------|------|
| **CPI** | $8 | $15 | +87% |
| **LTV** | $24 | $24 | 0% |
| **R (LTV/CPI)** | 3.0 | 1.6 | -47% |
| **盈利时间** | 7.5 个月 | **32 个月+** | 极长 |
| **垫资金额** | ~$800M | **$2,000M+** | — |

```
CPI 涨一倍，不是"利润少一半"，而是"从盈利变破产"。

临界点：CPI = LTV / (1+c) = LTV / 1.4
  CPI $8 时：LTV/1.4 = $17，CPI < $17 → 能跑
  CPI $15 时：LTV/1.4 = $17，CPI < $17 → 理论上还能跑，但 N 很长

实际决策中，考虑到 CPI 动态上涨和安全边际：
  建议 CPI < LTV / 2.5 才有足够空间
  CPI $8 时：LTV/2.5 = $9.6，CPI < $9.6 → 安全
  CPI $15 时：LTV/2.5 = $9.6，CPI > $9.6 → 危险
```

### SLG 12. SLG 实操案例

```

现金流：

  - 可以支撑到回本
```
