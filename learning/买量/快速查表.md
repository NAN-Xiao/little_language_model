# SLG 买量投放决策速查表

> 按时间节点（Day1 → Day90），逐项判断项目是否值得继续投入。
> 适用于中度 SLG（Last War / Whiteout Survival 级别），美国市场 Meta 渠道。

---

## 一、分阶段核心指标速查

### Day 1 — 次日留存判断

| 指标 | 止损 ❌ | 观察 ⚠️ | 可跑 ✅ | 优秀 🚀 |
|------|--------|--------|--------|--------|
| **D1 留存** | < 20% | 20-25% | 25-35% | > 35% |
| **CPI** | > $30 | $20-30 | $12-18 | < $12 |

**决策：**
```
D1 < 20% 且 CPI > $25 → 立刻止损，产品或素材有大问题
D1 > 30% 且 CPI < $15 → 继续观察 Day3/7
其他情况 → 等 Day3 数据再判断
```

---

### Day 3 — 早期留存验证

| 指标 | 止损 ❌ | 观察 ⚠️ | 可跑 ✅ | 优秀 🚀 |
|------|--------|--------|--------|--------|
| **D3 留存** | < 10% | 10-15% | 15-22% | > 22% |
| **D3 ROAS** | < 3% | 3-5% | 5-8% | > 8% |

**决策：**
```
D3 留存 < 12% → 大概率 D30 留存 < 4%，项目不成立
D3 ROAS < 4% → 最终 ROAS 很难超过 100%，谨慎
D3 ROAS > 6% + CPI < $15 → 有放量潜力，等 Day7
```

---

### Day 7 — 关键分水岭

| 指标 | 止损 ❌ | 观察 ⚠️ | 可跑 ✅ | 优秀 🚀 |
|------|--------|--------|--------|--------|
| **D7 留存** | < 8% | 8-12% | 12-18% | > 18% |
| **Day7 付费率** | < 2% | 2-3% | 3-5% | > 5% |
| **Day7 ARPPU** | < $35 | $35-50 | $50-70 | > $70 |
| **Day7 ROAS** | < 8% | 8-12% | 12-18% | > 18% |
| **CPI（中度SLG）** | > $25 | $18-25 | $12-18 | < $12 |

**决策：**
```
Day7 ROAS < 10% + CPI > $20 → 止损
Day7 ROAS 12-18% + D7 留存 > 12% → 继续跑，等 Day15
Day7 ROAS > 18% + CPI < $15 → 优秀，可加大预算
```

---

### Day 15 — 付费深度验证

| 指标 | 止损 ❌ | 观察 ⚠️ | 可跑 ✅ | 优秀 🚀 |
|------|--------|--------|--------|--------|
| **D15 留存** | < 4% | 4-6% | 6-10% | > 10% |
| **Day15 ROAS** | < 20% | 20-30% | 30-45% | > 45% |
| **首充率** | < 1% | 1-2% | 2-3.5% | > 3.5% |

**预测 Day30：**
```
Day15 ROAS × (1/0.5 ~ 1/0.6) ≈ Day30 ROAS
例：Day15 = 35% → Day30 ≈ 58-70%
```

**决策：**
```
Day15 ROAS < 25% → 最终很难达到 100%，考虑止损
Day15 ROAS 30-40% → 可跑，但盈利空间有限
Day15 ROAS > 45% → 大概率能盈利，等 Day30 确认
```

---

### Day 30 — 核心决策点 ⚡

| 指标 | 止损 ❌ | 观察 ⚠️ | 可跑 ✅ | 优秀 🚀 |
|------|--------|--------|--------|--------|
| **D30 留存** | < 3% | 3-5% | 5-8% | > 8% |
| **Day30 付费率** | < 3% | 3-4% | 4-6% | > 6% |
| **Day30 ARPPU** | < $45 | $45-60 | $60-85 | > $85 |
| **Day30 ROAS** | < 40% | 40-60% | 60-80% | > 80% |

**预测最终 ROAS（关键！）：**
```
Day30 ROAS × 2.5 ~ 3.5 = Day180 ROAS

速算：
  Day30 = 40% → Day180 ≈ 100-140%
  Day30 = 60% → Day180 ≈ 150-210%
  Day30 = 80% → Day180 ≈ 200-280%
```

**盈亏平衡判断（含运营成本 c=0.4）：**
```
公司盈利所需 Day180 ROAS = 140%
对应所需 Day30 ROAS = 140% / 3 = 47%

Day30 < 40% → 止损
Day30 40-50% → 微利，谨慎跑，严格控制 CPI
Day30 50-70% → 可跑，正常投放
Day30 > 70% → 放量，加大预算
```

---

### Day 60 — 中期验证

| 指标 | 止损 ❌ | 观察 ⚠️ | 可跑 ✅ | 优秀 🚀 |
|------|--------|--------|--------|--------|
| **Day60 ROAS** | < 70% | 70-90% | 90-120% | > 120% |
| **D60 留存** | < 2% | 2-3% | 3-5% | > 5% |

**预测最终：**
```
Day60 ROAS × 1.5 ~ 2.0 = Day180 ROAS

Day60 < 90% → 最终可能到不了 140%，风险高
Day60 > 100% → 单用户已回本，公司层面也快了
```

---

### Day 90 — 回本确认

| 指标 | 止损 ❌ | 观察 ⚠️ | 可跑 ✅ | 优秀 🚀 |
|------|--------|--------|--------|--------|
| **Day90 ROAS** | < 100% | 100-120% | 120-150% | > 150% |
| **D90 留存** | < 1.5% | 1.5-2.5% | 2.5-4% | > 4% |

**决策：**
```
Day90 < 100% → 单用户都没回本，止损
Day90 100-120% → 单用户回本，但公司层面还没盈利（要覆盖运营），继续观察
Day90 120-150% → 公司层面接近盈利，可维持投放
Day90 > 150% → 盈利确认，放量
```

## 二、综合决策矩阵（Day30 核心判断）

以 **Day30 ROAS** 和 **LTV/CPI (R)** 两个维度判断：

|  | R < 1.5 | R = 1.5-2 | R = 2-3 | R > 3 |
|--|---------|-----------|---------|-------|
| **Day30 < 40%** | ❌ 止损 | ❌ 止损 | ⚠️ 观察 | ⚠️ 观察 |
| **Day30 40-60%** | ❌ 止损 | ⚠️ 观察 | ✅ 可跑 | ✅ 放量 |
| **Day30 60-80%** | ⚠️ 观察 | ✅ 可跑 | ✅ 放量 | 🚀 大力放量 |
| **Day30 > 80%** | ✅ 可跑 | ✅ 放量 | 🚀 放量 | 🚀 暴利 |

**说明：**
- R = LTV/CPI，需结合留存曲线估算
- Day30 ROAS 是最可靠的早期预测指标
- 两者矛盾时（如 Day30 高但 R 低），以 Day30 为准（更反映当前素材/定向效率）

## 三、CPI 快速判断（美国 Meta，中度 SLG）

| CPI 区间 | 判断 | 行动 |
|----------|------|------|
| **< $10** | 🟢 极低 | 不管什么留存都可以试 |
| **$10-15** | 🟢 优秀 | 只要 D1 > 25% 就值得继续 |
| **$15-20** | 🟡 正常 | 需要 D7 留存 > 12% 才能跑 |
| **$20-25** | 🟠 偏高 | 需要 Day7 ROAS > 15% 才能继续 |
| **> $25** | 🔴 危险 | 除非 Day15 ROAS > 30%，否则止损 |

**CPI 与 LTV 匹配速算：**
```
CPI = $15，需要 R > 2 才能舒服跑 → LTV > $30
CPI = $20，需要 R > 2.5 → LTV > $50
CPI = $25，需要 R > 3 → LTV > $75
```

## 四、留存率判断链

```
D1 留存 → 预测 D7：D7 ≈ D1 × 0.45-0.55
D7 留存 → 预测 D30：D30 ≈ D7 × 0.40-0.50
D30 留存 → 预测 D90：D90 ≈ D30 × 0.35-0.45

例：D1 = 32% → D7 ≈ 14-18% → D30 ≈ 5.6-9% → D90 ≈ 2-4%
```

| D1 | D7 | D30 | 评级 |
|----|----|-----|------|
| > 35% | > 18% | > 8% | 🟢 优秀（Top 10%） |
| 28-35% | 12-18% | 5-8% | 🟡 良好（可跑） |
| 22-28% | 8-12% | 3-5% | 🟠 及格（需优化） |
| < 22% | < 8% | < 3% | 🔴 差（考虑止损） |

## 五、一句话决策口诀

```
Day1 看留存 + CPI，太差就止损
Day7 看付费 + ROAS，验证变现能力
Day15 看深度，预测最终
Day30 定生死，ROAS × 3 = 最终
Day90 确认回本，决定加不加量

底线：Day30 ROAS < 40% → 90%概率不盈利 → 止损
黄金：Day30 ROAS > 60% + CPI < $15 → 大概率赚钱 → 放量
```
