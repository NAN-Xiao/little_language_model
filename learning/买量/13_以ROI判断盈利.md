# 以 ROI 为入口判断盈利

> 已知 DayN 的 ROAS/ROI，如何判断项目最终能否盈利，以及盈利空间有多大。

---

## 一、核心逻辑

```
已知：DayN ROAS = X%
目标：预测 Day180 / Day365 ROAS
判断：最终 ROAS 能否覆盖 买量成本 + 运营成本
```

**关键公式：**

```
单用户回本：最终 ROAS > 100%
公司盈利：最终 ROAS > 100% × (1 + c)

其中 c = 运营成本 / 买量成本
```

---

## 二、用早期 ROI 预测最终 ROI

### 2.1 SLG 典型 ROI 增长规律

```
经验比值（基于大量 SLG 数据）：

Day7  / Day30  ≈ 0.20-0.30
Day15 / Day30  ≈ 0.45-0.60
Day30 / Day90  ≈ 0.45-0.55
Day90 / Day180 ≈ 0.60-0.75

即：
Day90  ROAS ≈ Day30 × (1.8-2.2)
Day180 ROAS ≈ Day30 × (2.5-3.5)
Day365 ROAS ≈ Day30 × (3.5-5.0)
```

### 2.2 预测速查表

| 已知 Day30 ROAS | 预测 Day90 | 预测 Day180 | 预测 Day365 |
|----------------|-----------|------------|------------|
| **30%** | 55-65% | 75-105% | 105-150% |
| **40%** | 70-85% | 100-140% | 140-200% |
| **50%** | 90-110% | 125-175% | 175-250% |
| **60%** | 110-130% | 150-210% | 210-300% |
| **80%** | 145-175% | 200-280% | 280-400% |

### 2.3 对数回归预测法

```
ROAS(t) = a × ln(t) + b

用两个已知点求 a, b：

示例：
  已知 Day7 ROAS = 12%，Day30 ROAS = 45%
  
  12 = a × ln(7) + b  →  12 = 1.946a + b
  45 = a × ln(30) + b →  45 = 3.401a + b
  
  解得：a = 33/1.455 = 22.68
       b = 12 - 1.946×22.68 = -32.13
  
  预测 Day180：
  ROAS(180) = 22.68 × 5.193 - 32.13 = 85.7%
```

## 三、判断能不能盈利

### 3.1 单用户层面

```
单用户回本条件：最终 ROAS > 100%

用 Day30 ROAS 快速判断：
  Day30 > 40% → 大概率最终 > 100%（能回本）
  Day30 < 30% → 大概率最终 < 100%（回不了本）
```

### 3.2 公司层面（关键！）

```
公司盈利条件：最终 ROAS > 100% × (1 + c)

示例：
  月买量 $500K，月运营 $200K
  c = 200/500 = 0.4
  
  需要最终 ROAS > 140% 才能公司层面盈利
  
  如果 Day30 ROAS = 60%
  预测 Day180 ROAS ≈ 150-210%
  → 150% > 140% → 勉强盈利
  → 安全边际很小，CPI 上涨就崩
```

### 3.3 盈亏平衡 ROI 表

| 运营成本比例 c | 单用户回本所需 ROAS | 公司盈利所需 ROAS |
|--------------|-------------------|-----------------|
| 0% | 100% | 100% |
| 20% | 100% | 120% |
| 40% | 100% | 140% |
| 60% | 100% | 160% |
| 100% | 100% | 200% |

---

## 四、盈利空间计算

### 4.1 盈利空间公式

```
盈利空间 = (预测最终 ROAS - 盈亏平衡 ROAS) / 盈亏平衡 ROAS

示例：
  Day30 ROAS = 60%
  预测 Day180 ROAS = 180%
  c = 0.4，盈亏平衡 = 140%
  
  盈利空间 = (180% - 140%) / 140% = 28.6%
  → 每投入 $1，最终赚回 $1.286（净利润 $0.286）
```

### 4.2 不同 ROI 对应的盈利空间

假设 c = 0.4（盈亏平衡 ROAS = 140%）：

| Day30 ROAS | 预测 Day180 | 能否盈利 | 盈利空间 | 决策 |
|-----------|------------|---------|---------|------|
| 30% | 75-105% | ❌ 不能 | — | 止损 |
| 40% | 100-140% | ⚠️ 勉强 | 0-7% | 谨慎/观察 |
| 50% | 125-175% | ✅ 能 | 7-25% | 可跑 |
| 60% | 150-210% | ✅ 能 | 28-50% | 放量 |
| 80% | 200-280% | ✅✅ 能 | 43-100% | 大力放量 |

---

## 五、快速决策流程

```
Step 1：拿到 Day30 ROAS
  ↓
Step 2：预测 Day180 ROAS ≈ Day30 × (2.5-3.5)
  ↓
Step 3：计算盈亏平衡 ROAS = 100% × (1 + c)
  ↓
Step 4：比较并决策

  预测 ROAS < 盈亏平衡
    → 不盈利，止损
    
  预测 ROAS = 盈亏平衡 × 1.0-1.2
    → 微利，谨慎跑，严格控制成本
    
  预测 ROAS = 盈亏平衡 × 1.2-1.5
    → 盈利，正常跑
    
  预测 ROAS > 盈亏平衡 × 1.5
    → 暴利，放量
```

## 六、实际案例

### 案例 1：Whiteout Survival（成功）

```
Day30 ROAS = 75%
c ≈ 0.3
盈亏平衡 ROAS = 130%

预测 Day180 ROAS = 75% × 3 = 225%
盈利空间 = (225% - 130%) / 130% = 73%

→ 决策：放量 ✅✅
```

### 案例 2：Last War 2025（崩盘）

```
Day30 ROAS = 35%
c ≈ 0.4
盈亏平衡 ROAS = 140%

预测 Day180 ROAS = 35% × 3 = 105%
105% < 140% → 不盈利

→ 决策：止损 ❌
（实际 2026 年 4 月停投）
```

### 案例 3：一般 SLG（微利）

```
Day30 ROAS = 50%
c = 0.4
盈亏平衡 ROAS = 140%

预测 Day180 ROAS = 50% × 3 = 150%
盈利空间 = (150% - 140%) / 140% = 7%

→ 决策：可以跑，但风险高
→ 必须严格控制 CPI，不能让它上涨
```

## 七、常见误区

```
❌ 误区 1：Day30 ROAS > 100% 就以为盈利了
   → 错！运营成本还没算
   → Day30 ROAS = 120%，c = 0.4
   → 盈亏平衡 = 140%
   → 120% < 140% → 实际亏损！

❌ 误区 2：只看单用户回本，不看公司层面
   → 单用户回本 ≠ 公司盈利
   → 公司每月都在买新用户，老用户赚的钱要填新用户的坑

❌ 误区 3：用 Day7 ROAS 直接判断
   → Day7 误差太大（±25%）
   → 至少等 Day15-30 再做决定

✅ 正确做法：
   Day30 ROAS → 预测 Day180 → 对比盈亏平衡 ROAS → 计算盈利空间 → 决策
```
