# 买量与游戏数据名词汇总

> 集中整理买量投放和游戏运营中的所有核心数据名词，方便快速查阅。

---

## 一、投放/买量指标

### 1.1 成本类

| 缩写     | 全称              | 中文         | 公式                   | 说明               |
| -------- | ----------------- | ------------ | ---------------------- | ------------------ |
| **CPI**  | Cost Per Install  | 单次安装成本 | 总花费 / 安装数        | 获取一个用户的成本 |
| **CPM**  | Cost Per Mille    | 千次展示成本 | 总花费 / (展示数/1000) | 每1000次展示的价格 |
| **CPC**  | Cost Per Click    | 单次点击成本 | 总花费 / 点击数        | 每次点击的价格     |
| **CPA**  | Cost Per Action   | 单次行动成本 | 总花费 / 行动数        | 按特定行为付费     |
| **CPP**  | Cost Per Purchase | 单次购买成本 | 总花费 / 购买次数      | AEO中衡量付费效率  |
| **oCPM** | Optimized CPM     | 优化千次展示 | —                      | Meta的自动出价模式 |

### 1.2 效果类

| 缩写     | 全称                 | 中文             | 公式                   | 说明               |
| -------- | -------------------- | ---------------- | ---------------------- | ------------------ |
| **CTR**  | Click Through Rate   | 点击率           | 点击数 / 展示数        | 广告吸引点击的能力 |
| **CVR**  | Conversion Rate      | 转化率           | 安装数 / 点击数        | 点击后安装的比例   |
| **IPM**  | Install Per Mille    | 千次展示安装数   | (安装数/展示数) × 1000 | 综合衡量素材效果   |
| **ROAS** | Return On Ad Spend   | 广告支出回报率   | 累计收入 / 买量成本    | 买量投入产出比     |
| **ROI**  | Return On Investment | 投资回报率       | (收入-成本) / 成本     | 整体投资回报率     |
| **LTV**  | Lifetime Value       | 用户生命周期价值 | ARPU × 平均生命周期    | 一个用户终身价值   |
| **RPD**  | Revenue Per Download | 下载用户收入     | 总收入 / 下载数        | 同 ARPU            |

### 1.3 平台/策略类

| 缩写     | 全称                       | 中文           | 说明                        |
| -------- | -------------------------- | -------------- | --------------------------- |
| **AEO**  | App Event Optimization     | 应用事件优化   | Meta 优化应用内事件的策略   |
| **VO**   | Value Optimization         | 价值优化       | Meta 优化用户付费金额的策略 |
| **A+AC** | Advantage+ App Campaign    | 全自动应用广告 | Meta 的自动化投放           |
| **UAC**  | Universal App Campaign     | 通用应用广告   | Google 的应用广告           |
| **ASA**  | Apple Search Ads           | 苹果搜索广告   | 在 App Store 搜索结果中投放 |
| **SDK**  | Software Development Kit   | 软件开发工具包 | 接入广告归因的代码包        |
| **MMP**  | Mobile Measurement Partner | 移动监测伙伴   | Adjust/AppsFlyer 等归因平台 |

## 二、游戏内运营指标

### 2.1 用户规模

| 缩写     | 全称                 | 中文       | 说明                 |
| -------- | -------------------- | ---------- | -------------------- |
| **DAU**  | Daily Active Users   | 日活跃用户 | 当日登录的用户数     |
| **WAU**  | Weekly Active Users  | 周活跃用户 | 7日内登录的用户数    |
| **MAU**  | Monthly Active Users | 月活跃用户 | 30日内登录的用户数   |
| **新增** | New Users            | —          | 当日首次安装的用户   |
| **回流** | Returning Users      | —          | 流失后重新登录的用户 |

### 2.2 留存率

| 缩写    | 全称             | 中文     | 公式                 | SLG 标准           |
| ------- | ---------------- | -------- | -------------------- | ------------------ |
| **D1**  | Day 1 Retention  | 次日留存 | Day1登录 / 首日新增  | >25%及格，>35%优秀 |
| **D3**  | Day 3 Retention  | 3日留存  | Day3登录 / 首日新增  | >15%及格，>20%优秀 |
| **D7**  | Day 7 Retention  | 7日留存  | Day7登录 / 首日新增  | >8%及格，>15%优秀  |
| **D15** | Day 15 Retention | 15日留存 | Day15登录 / 首日新增 | >5%及格，>8%优秀   |
| **D30** | Day 30 Retention | 30日留存 | Day30登录 / 首日新增 | >3%及格，>6%优秀   |

### 2.3 收入类

| 缩写         | 全称                            | 中文               | 公式                | 说明             |
| ------------ | ------------------------------- | ------------------ | ------------------- | ---------------- |
| **ARPU**     | Average Revenue Per User        | 每用户平均收入     | 总收入 / 总用户数   | 所有人（含白嫖） |
| **ARPPU**    | Average Revenue Per Paying User | 每付费用户平均收入 | 总收入 / 付费用户数 | 只算付费的人     |
| **ARPDAU**   | Average Revenue Per DAU         | 日活用户平均收入   | 日收入 / DAU        | 每日变现效率     |
| **付费率**   | Paying User Rate                | —                  | 付费人数 / 总人数   | SLG 通常 3-8%    |
| **首充率**   | First Purchase Rate             | —                  | 首充人数 / 总人数   | SLG 通常 1-3%    |
| **首充金额** | First Purchase Amount           | —                  | —                   | 首充礼包价格     |

### 2.4 用户分层

| 层级         | 定义          | 占比   | 贡献收入 | 说明            |
| ------------ | ------------- | ------ | -------- | --------------- |
| **白嫖/小R** | 付费 <$50     | 70-85% | 5-15%    | 基本不付费      |
| **中R**      | 付费 $50-500  | 10-20% | 15-30%   | 月卡/礼包       |
| **大R/鲸鱼** | 付费 >$500    | 2-5%   | 60-80%   | SLG核心收入来源 |
| **超R**      | 付费 >$10,000 | <1%    | 20-40%   | 服务器顶级玩家  |

### 2.5 SLG 特有指标

| 名词               | 说明                        | SLG 标准     |
| ------------------ | --------------------------- | ------------ |
| **联盟加入率**     | 加入联盟人数 / 注册数       | 40-70%       |
| **城堡等级达标率** | 达到指定等级的用户比例      | Day7: 30-50% |
| **联盟战参与率**   | 参与联盟战的人数比例        | >60%         |
| **首次付费时间**   | 安装到第一次付费的天数      | 3-7 天       |
| **7日付费率**      | Day7 付费人数 / 安装数      | 2-5%         |
| **30日付费率**     | Day30 付费人数 / 安装数     | 4-8%         |
| **大R占比**        | 付费>$500 的用户 / 付费用户 | 5-15%        |

## 三、关键公式关系

```
┌─────────────────────────────────────────────────────────┐
│                    核心公式链                            │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  CPI = CPM / (CTR × CVR × 1000)                         │
│                                                         │
│  IPM = CTR × CVR × 1000                                 │
│                                                         │
│  ARPU = ARPPU × 付费率                                   │
│                                                         │
│  ARPDAU = 日收入 / DAU                                  │
│                                                         │
│  LTV = ARPU × 平均生命周期                               │
│                                                         │
│  ROAS = 累计收入 / 买量成本                              │
│                                                         │
│  LTV/CPI = R（收入倍数）                                 │
│                                                         │
│  公司盈利时间 N = T×R / (R-1-c)                          │
│                                                         │
│  垫资 ≈ 月总支出 × (T + N) / 2                           │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 四、常见混淆概念

### 4.1 ARPU vs ARPPU

```
ARPU = 总收入 / 总用户数（含白嫖）
ARPPU = 总收入 / 付费用户数（只算付费的）

关系：ARPU = ARPPU × 付费率

示例：
  总用户 10,000 人，付费用户 300 人，总收入 $15,000
  ARPU = $15,000 / 10,000 = $1.5
  ARPPU = $15,000 / 300 = $50
  付费率 = 3%
  验证：$50 × 3% = $1.5 ✅
```

### 4.2 ARPU vs ARPDAU

```
ARPU = 总收入 / 总用户数（累计人均，含已流失用户）
ARPDAU = 日收入 / DAU（当日活跃的活人）

关系：ARPDAU ≈ ARPU × (留存衰减系数) —— 不精确，仅供参考

关键区别：
  ARPU 看的是"这辈子"——用户从安装到现在的累计付费
  ARPDAU 看的是"今天"——当天还在玩的人今天花了多少钱

实战意义：
  ARPDAU 高 + 留存低 = 割韭菜模式（Last War）
  ARPDAU 高 + 留存高 = 健康变现（Whiteout）
  ARPDAU 低 + 留存高 = 有潜力，还没找到付费点

示例：
  游戏 A：ARPDAU = $0.5，D30 留存 8% → 长线健康
  游戏 B：ARPDAU = $1.2，D30 留存 3% → 短期暴利但不可持续
```

### 4.3 ROAS vs ROI

```
ROAS = 收入 / 买量成本（只看买量）
ROI = (收入 - 总成本) / 总成本（看全部）

示例：
  收入 $200K，买量 $100K，运营 $50K
  ROAS = 200K / 100K = 200%（买量翻倍）
  ROI = (200K - 150K) / 150K = 33%（整体赚了33%）
```

### 4.4 CPI vs CPP

```
CPI = 获得一次安装的成本
CPP = 获得一次付费的成本

关系：CPP = CPI / 付费率

示例：
  CPI = $15，付费率 3%
  CPP = $15 / 3% = $50
```

### 4.5 AEO vs VO

```
AEO：优化"会付费的人"（0/1，有付费就行）
VO：优化"付大钱的人"（金额，要付得多）

区别：
  AEO 用户：首充 $5-30 的小R
  VO 用户：首充 $100-500 的大R
```
