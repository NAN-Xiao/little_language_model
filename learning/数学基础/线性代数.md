# 线性代数 —— 从基础到应用

> 线性代数是描述"多维空间"和"变换"的语言。深度学习中几乎所有数据都是向量和矩阵。

---

## 一、基本概念

### 1.1 标量（Scalar）

一个单独的数。

```
a = 5
b = 3.14
```

### 1.2 向量（Vector）

一组有顺序的数，排成一行或一列。

```
行向量:  [1, 2, 3]          (1, 3)

列向量:  [1]                (3, 1)
         [2]
         [3]
```

向量可以表示：一个点的坐标、一个物体的特征、一个词的嵌入向量。

### 1.3 矩阵（Matrix）

二维的数表，有行有列。

```
    第0列  第1列  第2列
    ┌────┬────┬────┐
第0行│ 1  │ 2  │ 3  │
    ├────┼────┼────┤
第1行│ 4  │ 5  │ 6  │
    └────┴────┴────┘

(2, 3)  ← 2行3列
```

### 1.4 张量（Tensor）

任意维度的数组。

| 维度 | 名称 | 例子 |
|------|------|------|
| 0维 | 标量 | `5` |
| 1维 | 向量 | `[1, 2, 3]` |
| 2维 | 矩阵 | `[[1,2],[3,4]]` |
| 3维 | 张量 | `[[[1,2],[3,4]], [[5,6],[7,8]]]` |

## 二、向量的运算

### 2.1 向量加法

对应位置相加，形状必须相同。

```
[1]     [4]     [1+4]     [5]
[2]  +  [5]  =  [2+5]  =  [7]
[3]     [6]     [3+6]     [9]
```

### 2.2 向量数乘

每个元素乘以标量。

```
    [1]     [2×1]     [2]
2 × [2]  =  [2×2]  =  [4]
    [3]     [2×3]     [6]
```

### 2.3 点积（内积）

对应位置相乘，然后全部加起来。

```
a = [1, 2, 3]
b = [4, 5, 6]

a · b = 1×4 + 2×5 + 3×6 = 4 + 10 + 18 = 32
```

**几何意义：** 点积越大，两个向量方向越接近。

### 2.4 外积

两个向量产生一个矩阵。

```
       [10, 20, 30]
[1]    [10,  20,  30]
[2] ×  [20,  40,  60]
[3]    [30,  60,  90]

(3,) ⊗ (3,) = (3, 3)
```

## 三、矩阵的基本运算

### 3.1 矩阵加法

对应位置相加，形状必须相同。

```
[1, 2]     [5, 6]     [6,  8]
[3, 4]  +  [7, 8]  =  [10, 12]
```

### 3.2 矩阵数乘

每个元素乘以标量。

```
    [1, 2]     [2, 4]
2 × [3, 4]  =  [6, 8]
```

### 3.3 矩阵乘法

行乘列求和。`(M, K) @ (K, N) = (M, N)`

```
[1, 2]     [5, 6]     [1×5+2×7, 1×6+2×8]     [19, 22]
[3, 4]  @  [7, 8]  =  [3×5+4×7, 3×6+4×8]  =  [43, 50]
```

### 3.4 转置

行列互换。

```
    [1, 2, 3]             [1, 4]
A = [4, 5, 6]     A.T =   [2, 5]
    (2, 3)                [3, 6]
                          (3, 2)
```

### 3.5 单位矩阵

对角线是1，其他是0。任何矩阵乘单位矩阵等于自己。

```
    [1, 0]
I = [0, 1]

[3, 4]   [1, 0]   [3, 4]
[5, 6] × [0, 1] = [5, 6]
```

## 四、行列式（Determinant）

### 4.1 行列式是什么

行列式是一个数，描述矩阵的"缩放能力"。

```
    |a  b|
|A| = |c  d| = ad - bc
```

**例子：**

```
    |2  3|
|A| = |1  4| = 2×4 - 3×1 = 8 - 3 = 5
```

### 4.2 几何意义

行列式的绝对值 = 面积（2维）或体积（3维）的缩放倍数。

```
行列式 = 0  →  矩阵把空间压扁了（不可逆）
行列式 > 0  →  矩阵把空间放大了
行列式 < 0  →  矩阵把空间翻转了
```

## 五、逆矩阵

### 5.1 逆矩阵是什么

逆矩阵是"撤销"操作。`A @ A⁻¹ = I`（单位矩阵）

```
如果 A 把 [x] 变成 [y]
那么 A⁻¹ 把 [y] 变回 [x]
```

### 5.2 求逆条件

只有行列式不为0的矩阵才有逆矩阵。

```
|A| ≠ 0  →  A 可逆，有逆矩阵 A⁻¹
|A| = 0  →  A 不可逆
```

## 六、特征值与特征向量

### 6.1 直观理解

矩阵是一种变换。有些向量被矩阵变换后，方向不变，只被拉伸了。

```
A × v = λ × v

v  →  特征向量（方向不变的向量）
λ  →  特征值（拉伸的倍数）
```

**例子：**

```
拉伸变换：
[2, 0]   [1]     [2]       [1]
[0, 2] × [0]  =  [0]  = 2 × [0]

v = [1, 0]  是特征向量
λ = 2       是特征值（拉伸了2倍）
```

### 6.2 用途

- PCA降维：保留特征值大的方向
-  pagerank：特征向量表示网页重要性

## 七、秩（Rank）

### 7.1 秩是什么

秩 = 矩阵中"真正独立"的行（或列）的个数。

```
[1, 2, 3]
[2, 4, 6]    ← 第2行 = 第1行×2，不独立
[1, 1, 1]

秩 = 2（第1行和第3行独立，第2行是多余的）
```

### 7.2 秩的意义

```
秩 = 矩阵能表示的信息维度
秩越小，信息越冗余
```

## 八、线性代数在深度学习中的应用

| 概念 | 深度学习中的对应 |
|------|----------------|
| 向量 | 词嵌入、特征向量 |
| 矩阵 | 权重矩阵、图像数据 |
| 矩阵乘法 | 神经网络的前向传播 |
| 转置 | Attention 中的 Q @ K.T |
| 点积 | 计算两个向量的相似度 |
| 特征值 | PCA降维、谱分析 |
