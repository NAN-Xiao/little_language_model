# 四元数 —— 零基础入门

> 四元数是描述3D旋转的数学工具。比欧拉角更优雅，比旋转矩阵更简洁。

---

## 一、为什么要学四元数

### 1.1 欧拉角的问题

用三个角度（俯仰、偏航、翻滚）表示旋转，直观但有问题。

**万向节死锁：**

```
飞机姿态：
  俯仰(pitch) = 90度（机头朝上）
  偏航(yaw) = 任意值
  翻滚(roll) = 任意值

当俯仰=90度时，偏航和翻滚变成同一个轴！
两个自由度丢失了一个，无法表示所有旋转。
```

### 1.2 旋转矩阵的问题

3×3旋转矩阵可以表示任意旋转，但需要9个数，而且容易积累数值误差。

### 1.3 四元数的优势

| 方式 | 参数个数 | 万向节死锁 | 插值 |
|------|---------|-----------|------|
| 欧拉角 | 3 | 有 | 难 |
| 旋转矩阵 | 9 | 无 | 难 |
| **四元数** | **4** | **无** | **容易** |

### 1.4 四元数到底是什么

**简单来说：四元数就是描述笛卡尔坐标系中"怎么旋转"的工具。**

```
欧拉角说：绕X轴转30度，绕Y轴转45度，绕Z轴转60度
         ↑ 三个角度，容易理解，但会死锁

旋转矩阵说：用9个数描述旋转
           ↑ 无死锁，但9个数太冗余

四元数说：用4个数 [w, x, y, z] 描述旋转
         ↑ 无死锁、4个数、还能平滑插值
```

**严格来说：**

```
单位四元数（长度=1）→ 纯旋转（最常用）
非单位四元数      → 旋转 + 缩放
```

实际应用中（3D图形、机器人、姿态估计），四元数几乎总是单位四元数，专门用来表示旋转。

## 二、从复数到四元数

### 2.1 复数回顾

复数 = 实部 + 虚部

```
z = a + bi

其中 i² = -1

复数可以表示2D旋转：
乘以一个复数 = 旋转 + 缩放
```

### 2.2 扩展到3D

数学家哈密顿想：能不能用类似复数的东西表示3D旋转？

他花了15年，终于在1843年发现了四元数。

**关键突破：需要三个虚数单位，而不是一个。**

```
q = a + bi + cj + dk

i² = j² = k² = ijk = -1
```

**注意：i、j、k 本身不是 -1，它们的平方等于 -1。**

```
i² = -1  →  i = √-1   (和复数里的 i 一样)
j² = -1  →  j = √-1
k² = -1  →  k = √-1

而且它们之间还有乘法关系：
  i × j = k     j × k = i     k × i = j
  j × i = -k    k × j = -i    i × k = -j
```

## 三、四元数的结构

### 3.1 组成部分

```
q = w + xi + yj + zk

w  → 实部（标量部分）
xi + yj + zk  → 虚部（向量部分）
```

也可以写成：

```
q = [w, (x, y, z)]

w = cos(θ/2)           ← 旋转角的一半的余弦
(x, y, z) = sin(θ/2) × (ux, uy, uz)   ← 旋转轴×sin(半角)
```

### 3.2 单位四元数

表示旋转的四元数长度必须为1：

```
|q| = √(w² + x² + y² + z²) = 1
```

**验证：**

```
q = [cos(45°), sin(45°) × (0, 0, 1)]
  = [0.707, (0, 0, 0.707)]

|q| = √(0.707² + 0 + 0 + 0.707²)
    = √(0.5 + 0.5)
    = √1 = 1  ✓
```

### 3.3 旋转轴怎么确定

旋转轴不是算出来的，是根据**应用场景**确定的。常见情况：

**情况1：绕世界坐标轴旋转**

```
绕X轴转：u = (1, 0, 0)
绕Y轴转：u = (0, 1, 0)
绕Z轴转：u = (0, 0, 1)

例子：地球自转 → 绕Y轴
```

**情况2：绕物体自身的轴旋转**

```
飞机姿态：
  机头方向 = 自身X轴
  机翼方向 = 自身Y轴
  机顶方向 = 自身Z轴

俯仰 = 绕机翼轴（Y轴）转
偏航 = 绕机顶轴（Z轴）转
翻滚 = 绕机头轴（X轴）转
```

**情况3：从一个方向转到另一个方向**

```
起点向量 a = (1, 0, 0)   ← 面朝前
终点向量 b = (0, 1, 0)   ← 面朝右

旋转轴 = a × b（叉积）
       = (0, 0, 1)        ← 绕Z轴转90度

旋转角 = arccos(a·b / (|a||b|))
       = arccos(0) = 90°
```

**情况4：从四元数中提取旋转轴**

```
q = [w, (x, y, z)]

旋转角 θ = 2 × arccos(w)
旋转轴 u = (x, y, z) / sin(θ/2)

例子：q = [0.707, (0, 0, 0.707)]
  θ = 2 × arccos(0.707) = 2 × 45° = 90°
  u = (0, 0, 0.707) / sin(45°) = (0, 0, 0.707) / 0.707 = (0, 0, 1)
  → 绕Z轴转90度
```

### 3.4 世界坐标旋转 vs 局部坐标旋转

**核心区别：轴是固定的还是跟着物体转的。**

```
世界坐标系（全局，不动）：
  Y
  ↑
  |    飞机的局部坐标系（跟着飞机转）：
  |      ↑ Y'（机顶方向，始终垂直于机翼）
  |     /
  |    /   X'（机头方向）
  |   /
  |  Z'（机翼方向）
  └────────→ X
   /
  Z
```

**例子：先俯仰90度，再偏航90度**

```
飞机初始姿态：面朝前（沿X轴）

【世界坐标旋转】
Step 1: 绕世界X轴俯仰90度（机头朝上）
Step 2: 绕世界Y轴偏航90度

结果：机头朝上了，然后绕世界Y轴转 → 机头朝侧面了

【局部坐标旋转】
Step 1: 绕自身X轴俯仰90度（机头朝上）
Step 2: 绕自身Y轴偏航90度（注意：此时自身Y轴已经变了！）

结果：机头朝上后，绕"新的Y轴"（此时指向世界Z方向）转 → 机头朝后了
```

**两种顺序结果完全不同！**

| | 世界坐标旋转 | 局部坐标旋转 |
|---|------------|------------|
| 旋转轴 | 固定不动 | 随物体一起转 |
| 先X后Y | 先绕世界X转，再绕世界Y转 | 先绕自身X转，再绕自身Y转 |
| 结果 | 机头朝侧面 | 机头朝后 |

**四元数中的区别：**

```
q_X = 绕X轴转的旋转
q_Y = 绕Y轴转的旋转

世界坐标系下（从右到左乘）：
  q_total = q_Y × q_X

局部坐标系下（从左到右乘）：
  q_total = q_X × q_Y

注意顺序相反！
```

### 3.5 如何构造一个旋转四元数

**公式：**

```
旋转轴：u = (ux, uy, uz)   （单位向量，长度为1）
旋转角：θ

q = [cos(θ/2), sin(θ/2) × (ux, uy, uz)]
     ↑                    ↑
   实部                虚部
```

**具体例子：绕 Z 轴旋转 45 度**

```
旋转轴：Z 轴 = (0, 0, 1)
旋转角：θ = 45°

Step 1: 计算半角
  θ/2 = 22.5°

Step 2: 计算实部
  w = cos(22.5°) = 0.924

Step 3: 计算虚部
  sin(22.5°) = 0.383
  (x, y, z) = 0.383 × (0, 0, 1) = (0, 0, 0.383)

Step 4: 组装
  q = [0.924, (0, 0, 0.383)]

验证：√(0.924² + 0.383²) = √(0.854 + 0.147) = √1.001 ≈ 1  ✓
```

**如果已有旋转，再叠加45度：**

```
已有旋转：q1 = [1, (0, 0, 0)]   ← 无旋转
再转45度：q_delta = [0.924, (0, 0, 0.383)]

新旋转 = q_delta × q1
       = [0.924, (0, 0, 0.383)] × [1, (0, 0, 0)]
       = [0.924, (0, 0, 0.383)]
```

**绕 X 轴旋转 30 度：**

```
旋转轴：X 轴 = (1, 0, 0)
旋转角：θ = 30°

w = cos(15°) = 0.966
(x, y, z) = sin(15°) × (1, 0, 0) = (0.259, 0, 0)

q = [0.966, (0.259, 0, 0)]
```

**绕任意轴旋转：**

```
旋转轴：u = (0.6, 0.8, 0)   （斜着的轴，长度为1）
旋转角：θ = 90°

w = cos(45°) = 0.707
(x, y, z) = sin(45°) × (0.6, 0.8, 0) = (0.424, 0.566, 0)

q = [0.707, (0.424, 0.566, 0)]
```

## 四、四元数的基本运算

### 4.1 加法

对应分量相加。

```
q1 = [1, (2, 3, 4)]
q2 = [5, (6, 7, 8)]

q1 + q2 = [1+5, (2+6, 3+7, 4+8)]
        = [6, (8, 10, 12)]
```

### 4.2 数乘

每个分量乘以标量。

```
2 × q = [2w, (2x, 2y, 2z)]
```

### 4.3 乘法（哈密顿积）

四元数乘法是核心，不满足交换律！

```
q1 = [w1, v1]    (v1 = (x1, y1, z1))
q2 = [w2, v2]

q1 × q2 = [w1w2 - v1·v2, w1v2 + w2v1 + v1×v2]

其中：
  v1·v2  = 点积
  v1×v2  = 叉积
```

**具体展开：**

```
(w1 + x1i + y1j + z1k) × (w2 + x2i + y2j + z2k)

= (w1w2 - x1x2 - y1y2 - z1z2)
+ (w1x2 + x1w2 + y1z2 - z1y2)i
+ (w1y2 - x1z2 + y1w2 + z1x2)j
+ (w1z2 + x1y2 - y1x2 + z1w2)k
```

**为什么四元数乘法这么复杂？**

因为虚数单位之间有关系：

```
i × j = k     j × k = i     k × i = j
j × i = -k    k × j = -i    i × k = -j

注意：i × j ≠ j × i，所以四元数乘法不满足交换律！
```

### 4.4 共轭

```
q = [w, (x, y, z)]
q* = [w, (-x, -y, -z)]

虚部取反，实部不变。
```

### 4.5 逆

```
q⁻¹ = q* / |q|²

对于单位四元数（|q|=1）：
q⁻¹ = q*
```

## 五、四元数表示3D旋转

### 5.1 旋转公式

用四元数 `q` 旋转一个向量 `v`：

```
v' = q × [0, v] × q⁻¹

其中：
  [0, v] 是把3D向量写成纯虚四元数（实部为0）
  q 是单位四元数
```

**因为 q 是单位四元数，q⁻¹ = q***

所以：

```
v' = q × [0, v] × q*
```

### 5.2 具体例子

**问题：绕 Z 轴旋转 90 度，向量 [1, 0, 0] 变成什么？**

```
Step 1: 构造旋转四元数

旋转轴：u = (0, 0, 1)   (Z轴)
旋转角：θ = 90°

w = cos(45°) = 0.707
(x, y, z) = sin(45°) × (0, 0, 1) = (0, 0, 0.707)

q = [0.707, (0, 0, 0.707)]
```

```
Step 2: 把向量写成纯虚四元数

v = [1, 0, 0]
[0, v] = [0, (1, 0, 0)]
```

```
Step 3: 计算 q × [0, v]

q       = [0.707, (0,    0,    0.707)]
[0, v]  = [0,     (1,    0,    0    )]

q × [0,v] = [0.707×0 - (0,0,0.707)·(1,0,0),
             0.707×(1,0,0) + 0×(0,0,0.707) + (0,0,0.707)×(1,0,0)]

点积: (0,0,0.707)·(1,0,0) = 0
叉积: (0,0,0.707)×(1,0,0) = (0×0-0.707×0, 0.707×1-0×0, 0×0-0×1)
                           = (0, 0.707, 0)

结果: [0, (0.707, 0, 0) + (0, 0.707, 0)]
    = [0, (0.707, 0.707, 0)]
```

```
Step 4: 计算 [0, (0.707, 0.707, 0)] × q*

q* = [0.707, (0, 0, -0.707)]

[0, (0.707, 0.707, 0)] × [0.707, (0, 0, -0.707)]

= [0×0.707 - (0.707,0.707,0)·(0,0,-0.707),
   0×(0,0,-0.707) + 0.707×(0.707,0.707,0) + (0.707,0.707,0)×(0,0,-0.707)]

点积: (0.707,0.707,0)·(0,0,-0.707) = 0

叉积:
  (0.707,0.707,0) × (0,0,-0.707)
= (0.707×(-0.707) - 0×0, 0×0 - 0.707×(-0.707), 0.707×0 - 0.707×0)
= (-0.5, 0.5, 0)

结果: [0, 0.707×(0.707,0.707,0) + (-0.5, 0.5, 0)]
    = [0, (0.5, 0.5, 0) + (-0.5, 0.5, 0)]
    = [0, (0, 1, 0)]
```

**答案：v' = [0, 1, 0]**

```
原向量 [1, 0, 0]（指向X轴）
绕Z轴转90度后 → [0, 1, 0]（指向Y轴）

     Y
     ↑
     |    ● (0,1)  ← 旋转后
     |   /
     |  /
     | /
     ●──────→ X
   (1,0)  ← 旋转前
```

### 5.3 多个旋转的复合

```
先绕X轴转30度，再绕Y轴转45度

q_total = q_Y(45°) × q_X(30°)

注意顺序：右边的是先做的旋转！
```

## 六、四元数插值（Slerp）

### 6.1 为什么需要插值

两个旋转之间平滑过渡。比如动画中角色从姿势A转到姿势B。

### 6.2 相对旋转：两个旋转的"差值"

**问题：** 已知旧旋转 q1 和新旋转 q2，从 q1 到 q2 需要再转多少？

**答案：可以！这个"差值"也是一个四元数。**

```
q_delta = q2 × q1⁻¹

其中：
  q1    = 旧旋转
  q2    = 新旋转
  q1⁻¹  = q1 的逆（单位四元数的逆 = 共轭）

验证：q_delta × q1 = q2 × q1⁻¹ × q1 = q2  ✓
```

**具体例子：**

```
q1 = [1, (0, 0, 0)]        ← 面朝前（0度）
q2 = [0.707, (0, 0.707, 0)] ← 面朝右（90度）

q1 是单位四元数，q1⁻¹ = q1* = [1, (0, 0, 0)]

q_delta = q2 × q1⁻¹
        = [0.707, (0, 0.707, 0)] × [1, (0, 0, 0)]
        = [0.707, (0, 0.707, 0)]

q_delta 表示：绕Y轴转90度

含义：从面朝前到面朝右，需要再绕Y轴转90度
```

**另一个例子：**

```
q1 = [0.707, (0, 0, 0.707)]   ← 面朝左（绕Z轴90度）
q2 = [0.707, (0, 0, 0.707)]   ← 还是面朝左（和q1相同）

q_delta = q2 × q1⁻¹ = [1, (0, 0, 0)]

q_delta 是单位四元数（无旋转）
含义：两个旋转相同，不需要再转
```

**所以：q_delta 就是两个旋转之间的"差值"，它本身也是一个四元数。**

### 6.3 具体例子：从面朝前转到面朝右

**问题：** 一个角色从面朝前方（q1）转到面朝右方（q2），中间怎么过渡？

```
初始姿态 q1：面朝前方（绕Y轴0度）
  q1 = [1, (0, 0, 0)]

目标姿态 q2：面朝右方（绕Y轴90度）
  q2 = [0.707, (0, 0.707, 0)]

t = 0.5：中间姿态
```

**Step 1: 计算 q1 和 q2 的夹角 θ**

```
cos(θ) = q1·q2   （四元数点积）

q1·q2 = 1×0.707 + 0×0 + 0×0.707 + 0×0 = 0.707

θ = arccos(0.707) = 45°
```

**Step 2: 代入 Slerp 公式**

```
t = 0.5，θ = 45°

sin((1-t)θ) = sin(0.5 × 45°) = sin(22.5°) = 0.383
sin(tθ)     = sin(0.5 × 45°) = sin(22.5°) = 0.383
sin(θ)      = sin(45°) = 0.707

Slerp = (0.383/0.707) × q1 + (0.383/0.707) × q2
      = 0.542 × [1, (0, 0, 0)] + 0.542 × [0.707, (0, 0.707, 0)]
      = [0.542, (0, 0, 0)] + [0.383, (0, 0.383, 0)]
      = [0.925, (0, 0.383, 0)]

验证长度：√(0.925² + 0.383²) = √(0.856 + 0.147) = √1.003 ≈ 1  ✓
```

**Step 3: 这个结果表示什么？**

```
q_mid = [0.925, (0, 0.383, 0)]

w = 0.925 = cos(θ/2)
θ/2 = arccos(0.925) ≈ 22.5°
θ = 45°

这是绕Y轴转45度的姿态！正好是0度和90度的正中间。
```

### 6.3 球面线性插值（Slerp）

```
Slerp(q1, q2, t) = (sin((1-t)θ) / sin(θ)) × q1 + (sin(tθ) / sin(θ)) × q2

其中 θ 是 q1 和 q2 之间的夹角，t 从 0 到 1

t=0: 结果是 q1
t=1: 结果是 q2
t=0.5: 中间旋转
```

**为什么不用普通线性插值？**

```
普通线性插值：q = (1-t)×q1 + t×q2
              ↓
         走直线，穿过球内部
         长度会变短（不再是单位四元数）
         旋转速度不均匀（先快后慢或先慢后快）

Slerp：沿着单位球面的大圆弧走
       ↓
  长度始终保持为1
  旋转速度恒定（匀速旋转）
```

**类比：**

```
地球表面两个点：
  线性插值 = 穿过地心的隧道（不对！）
  Slerp    = 沿着地球表面走大圆航线（正确！）
```

## 七、四元数 ↔ 欧拉角 ↔ 旋转矩阵

### 7.1 四元数转旋转矩阵

```
q = [w, (x, y, z)]

R = [1-2y²-2z²,   2xy-2wz,     2xz+2wy  ]
    [2xy+2wz,     1-2x²-2z²,   2yz-2wx  ]
    [2xz-2wy,     2yz+2wx,     1-2x²-2y²]
```

### 7.2 旋转矩阵转四元数

```
从旋转矩阵 R 中提取四元数：

w = √(1 + R[0,0] + R[1,1] + R[2,2]) / 2
x = (R[2,1] - R[1,2]) / (4w)
y = (R[0,2] - R[2,0]) / (4w)
z = (R[1,0] - R[0,1]) / (4w)
```

## 八、总结

| 概念 | 一句话 |
|------|--------|
| 四元数 | `q = w + xi + yj + dk`，4个数表示3D旋转 |
| 单位四元数 | 长度=1，用于表示旋转 |
| 旋转公式 | `v' = q × [0,v] × q*` |
| 乘法不满足交换律 | 旋转顺序很重要！ |
| 优势 | 无万向节死锁、4个数、插值容易 |
