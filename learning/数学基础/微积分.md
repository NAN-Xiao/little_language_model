# 微积分 —— 深度学习中的核心工具

> 不需要精通所有微积分。深度学习只用其中一小部分：**导数、梯度、链式法则**。

---

## 一、导数（Derivative）

### 1.1 导数的直觉

导数 = 变化率 = 斜率

```
f(x) = x²

在 x=3 处的导数 = 6
含义：当 x=3 时，x 每增加 1，f(x) 大约增加 6
```

### 1.2 导数怎么算

**方法一：极限定义（几何直觉）**

```
导数 = 切线斜率 = Δy / Δx 当 Δx 趋近于 0

f(x) = x²

在 x=3 处：
  f(3) = 9
  f(3.001) = 9.006001

  Δx = 0.001
  Δy = 9.006001 - 9 = 0.006001

  斜率 ≈ Δy / Δx = 0.006001 / 0.001 = 6.001 ≈ 6
```

**方法二：求导法则（实际计算）**

**幂函数法则：对 xⁿ 求导，分两步**

"指数掉下来"的意思：原来写在 x 上方的指数，求导后移到 x 的前面当乘数。

```
求导前:   x³        求导后:   3x²
           ↑                     ↑
        指数在上方             指数移到前面当系数

可视化:
  ³              3
 x      →      x²

"3"从 x 的头顶，掉到了 x 的前面
```

**公式：**

```
步骤1: 指数 n 移到前面当系数
步骤2: 指数减 1

f(x) = xⁿ  →  f'(x) = n × xⁿ⁻¹
```

**例子：**

```
f(x) = x³

步骤1: 指数 3 掉下来 → 3 × x
步骤2: 指数 3 减 1  → 3 × x²

f'(x) = 3x²
```

```
f(x) = x²

步骤1: 指数 2 掉下来 → 2 × x
步骤2: 指数 2 减 1  → 2 × x¹ = 2x

f'(x) = 2x

在 x=3 处：
f'(3) = 2 × 3 = 6
```

```
f(x) = x      (x = x¹)

步骤1: 指数 1 掉下来 → 1 × x
步骤2: 指数 1 减 1  → 1 × x⁰ = 1 × 1 = 1

f'(x) = 1     (斜率永远是1，一条45度直线)
```

```
f(x) = 5      (常数，可以看作 5 × x⁰)

步骤1: 指数 0 掉下来 → 0 × 5 × x
步骤2: 指数 0 减 1  → 0 × x⁻¹ = 0

f'(x) = 0     (常数不变化，变化率为0)
```

### 1.3 几何意义 —— 导数为什么等于斜率

**斜率 = 陡峭程度 = 竖直变化 / 水平变化**

```
       ↑
       |     ／
       |    ／
       |   ／    ← 这条线很陡，斜率大
       |  ／
       |／
       └──────→

       ↑
       |  －－－－
       |         ← 这条线很平，斜率小
       └──────→
```

**用 f(x) = x² 举例，看不同点的斜率：**

```
        ↑
    9   |        ● (3,9)
        |       ／
    4   |      ● (2,4)
        |     ／
    1   |    ● (1,1)
        |   ／
    0   |  ● (0,0)
        |／
        └──────→
          0 1 2 3
```

在 `x=3, y=9` 处画一条切线：

```
       ↑
    9  |        ●
       |       ／|
    6  |      ／ |  Δy = 6  (竖直变化)
       |     ／  |
    3  |    ／   |
       |   ／    |
    0  |  ●──────┘
       |   1     ← Δx = 1 (水平变化)
       └────────→

切线斜率 = Δy / Δx = 6 / 1 = 6

而 f'(3) = 2×3 = 6  ← 导数正好等于切线斜率
```

**不同点的斜率对比：**

| 点 | 切线斜率 | 导数 f'(x)=2x | 直观感受 |
|----|---------|---------------|---------|
| x=0 | 0 | 0 | 最低点，切线水平 |
| x=1 | 2 | 2 | 比较平缓 |
| x=2 | 4 | 4 | 比较陡 |
| x=3 | 6 | 6 | 很陡 |

**结论：导数 = 函数在某一点处切线的斜率**

### 1.4 深度学习中的应用

- 损失函数对权重的导数 = **梯度**
- 梯度告诉权重该怎么调整才能降低损失

## 二、常见函数的导数

| 函数 | 导数 | 深度学习中哪里用 |
|------|------|----------------|
| `c`（常数）| `0` | — |
| `x` | `1` | 线性层 |
| `x²` | `2x` | 平方损失 |
| `eˣ` | `eˣ` | Softmax 的底层 |
| `ln(x)` | `1/x` | 交叉熵损失 |

### 激活函数的导数

| 激活函数 | 公式 | 导数 | 哪里用 |
|---------|------|------|--------|
| ReLU | `max(0, x)` | `x>0: 1, x≤0: 0` | 隐藏层最常用 |
| Sigmoid | `1/(1+e⁻ˣ)` | `σ(x)(1-σ(x))` | 二分类输出 |
| Tanh | `(eˣ-e⁻ˣ)/(eˣ+e⁻ˣ)` | `1-tanh²(x)` | RNN中常用 |

## 三、链式法则（Chain Rule）

### 3.1 核心思想

**复合函数的导数 = 外层导数 × 内层导数**

```
如果 y = f(g(x))

dy/dx = dy/dg × dg/dx
         ↑         ↑
      外层导数   内层导数
```

### 3.2 例子

```
y = (2x + 1)³

外层：u³    → 导数 = 3u²
内层：2x+1  → 导数 = 2

y' = 3(2x+1)² × 2 = 6(2x+1)²
```

### 3.3 为什么链式法则是反向传播的灵魂

```
损失 L → 线性层2 → 激活层 → 线性层1 → 输入

反向传播时：
  dL/dW₂ = dL/d激活 × d激活/d线性2 × d线性2/dW₂
  dL/dW₁ = dL/d激活 × ... × d线性2/d激活 × d激活/d线性1 × d线性1/dW₁

每一层的梯度都"乘上"后面所有层的导数，一层一层传回去。
```

## 四、偏导数与梯度

### 4.1 导数 vs 偏导数

**导数**：函数只有一个变量

```
f(x) = x²

只有一个 x，对 x 求导就是导数：
f'(x) = 2x
```

**偏导数**：函数有多个变量，对**其中一个**求导，其他当作常数

```
f(x, y) = x² + 3xy + y²

有两个变量 x 和 y：
- 对 x 求偏导（把 y 当常数）：∂f/∂x = 2x + 3y
- 对 y 求偏导（把 x 当常数）：∂f/∂y = 3x + 2y
```

| | 导数 | 偏导数 |
|---|------|--------|
| 变量个数 | 1个 | 多个 |
| 符号 | f'(x) 或 df/dx | ∂f/∂x |
| 其他变量 | 没有 | 当作常数 |

**例子：对 x 求偏导时，y 怎么当常数**

```
f(x, y) = x² + 3xy + y²

把 y=2 代入（当成常数2）：
f(x, 2) = x² + 3x×2 + 2² = x² + 6x + 4

现在只有一个变量 x 了，求导：
df/dx = 2x + 6

而 ∂f/∂x = 2x + 3y，当 y=2 时：2x + 3×2 = 2x + 6
结果完全一样！
```

**所以：偏导数 = 假装其他变量是常数，然后求导。**

### 4.2 偏导数的计算 —— 一步一步拆解

```
f(x, y) = x² + 3xy + y²
```

**对 x 求偏导 ∂f/∂x（y 当常数）：**

**第一项 x²：**
- 正常求导，幂函数法则：x² → 2x

**第二项 3xy（关键是这里）：**

对 x 求导时，y 是常数。

```
3xy = (3y) × x

      ↑       ↑
    常数     变量

(常数) × x 的导数 = 常数

所以：3xy → 3y
```

验证：假设 y = 5（常数）
```
3xy = 3 × 5 × x = 15x
15x 的导数 = 15
而 3y = 3 × 5 = 15  ✓
```

**第三项 y²：**
- y 是常数，y² 也是常数，常数导数为 0

**汇总：**
```
∂f/∂x = 2x + 3y + 0 = 2x + 3y
```

**对 y 求偏导 ∂f/∂y（x 当常数）：**

**第一项 x²：**
- 对 y 求导时，x 是常数，所以 x² 也是常数
- 常数的导数为 0
- **注意：x² → 2x 是对 x 求导的结果，不是对 y！**

验证：假设 x = 4（常数）
```
x² = 4² = 16
16 是常数，导数 = 0  ✓
```

**第二项 3xy：**

对 y 求导时，x 是常数。

```
3xy = (3x) × y

      ↑       ↑
    常数     变量

(常数) × y 的导数 = 常数

所以：3xy → 3x
```

**第三项 y²：**
- y 是变量，正常求导：y² → 2y

**汇总：**
```
∂f/∂y = 0 + 3x + 2y = 3x + 2y
```

**代入具体数值验证：**

在点 (x=2, y=3) 处：

```
∂f/∂x = 2×2 + 3×3 = 4 + 9 = 13
∂f/∂y = 3×2 + 2×3 = 6 + 6 = 12
```

含义：在 (2,3) 这个点，x 每增加 1，f 大约增加 13；y 每增加 1，f 大约增加 12。

### 4.2 梯度（Gradient）

**梯度 = 把所有偏导数按顺序放在一起，组成一个向量**

```
∇f = [∂f/∂x, ∂f/∂y]

     ↑        ↑
   x方向的   y方向的
   变化率    变化率
```

**具体例子：**

用上面的 `f(x, y) = x² + 3xy + y²`，在点 `(2, 3)` 处：

```
∂f/∂x = 13    ← x 方向的变化率
∂f/∂y = 12    ← y 方向的变化率

梯度 ∇f = [13, 12]
```

**梯度是一个向量，有方向和大小：**

```
      ↑y
      |    ∇f = [13, 12]
      |     ↗
      |    ╱
      |   ╱   方向：往右上方（x和y都增加）
      |  ╱    大小：√(13² + 12²) ≈ 17.7
      | ╱
      └────────→x
           (2,3)
```

**梯度的含义：**

在点 `(2, 3)` 处，如果沿着 `[13, 12]` 这个方向移动，`f(x,y)` 增长最快。

**负梯度（梯度下降用的）：**

```
-∇f = [-13, -12]

方向：往左下方（x和y都减小）
含义：f(x,y) 下降最快的方向
```

**为什么深度学习用负梯度？**

```
损失函数 L 对权重 W 求梯度：
  ∇L = [∂L/∂W₁, ∂L/∂W₂, ..., ∂L/∂Wₙ]

梯度下降更新：
  W_new = W_old - lr × ∇L
              ↑
        往负梯度方向走，损失下降最快
```

**类比：小球滚下山**

```
        损失函数（山的高度）
           ∧
          ╱ ╲
         ╱   ╲
        ╱     ╲
       ╱   ●   ╲  ← 当前位置
      ╱    ↓    ╲   梯度 = 最陡的上坡方向
     ╱     ↓     ╲  负梯度 = 最陡的下坡方向
    ╱──────╱╲─────╲

优化器让小球沿着最陡的下坡方向滚，直到谷底（损失最小）
```

### 4.3 深度学习中的应用

```
模型有 1亿个参数 → 损失函数有 1亿个偏导数

梯度下降：
  W_new = W_old - lr × ∇Loss

lr = 学习率，∇Loss = 损失对所有参数的梯度
```

## 五、总结

| 概念 | 一句话解释 | 在深度学习中 |
|------|-----------|------------|
| 导数 | 变化率 | 损失函数的变化率 |
| 链式法则 | 复合函数求导，逐层相乘 | **反向传播的核心** |
| 偏导数 | 多变量中对一个变量求导 | 损失对单个权重的导数 |
| 梯度 | 所有偏导数组成的向量 | 告诉优化器怎么更新权重 |

**深度学习只需要这四样东西。**

## 六、微积分的其他应用

微积分不只是深度学习的工具，它是描述**变化**的通用语言。任何涉及"变化率"、"累积"、"优化"的领域都会用到。

---

### 6.1 物理学

**运动学：速度是位移的导数，加速度是速度的导数**

```
s(t) = 位移（随时间变化的函数）
v(t) = ds/dt = 速度（位移的变化率）
a(t) = dv/dt = 加速度（速度的变化率）

例子：自由落体
  s(t) = ½gt²    （位移）
  v(t) = gt      （速度，位移的导数）
  a(t) = g       （加速度，速度的导数，重力加速度 9.8m/s²）
```

**牛顿第二定律：F = ma**

```
力 = 质量 × 加速度
F = m × d²s/dt²

知道力，就能算出物体的运动轨迹。
```

**电磁学：麦克斯韦方程组**

```
电场和磁场都是用微分方程描述的：
  ∇·E = ρ/ε₀        （电场散度 = 电荷密度）
  ∇×E = -∂B/∂t      （电场旋度 = 磁场变化率）
  ∇·B = 0            （磁场散度 = 0）
  ∇×B = μ₀J + μ₀ε₀∂E/∂t  （磁场旋度 = 电流 + 电场变化率）

所有电磁现象（光、电、磁）都包含在这四个方程中。
```

**热力学：热量传递**

```
热量传导方程：∂u/∂t = α∇²u

u = 温度分布
∂u/∂t = 温度随时间的变化率
∇²u = 温度的空间二阶导数（温度的"曲率"）

含义：温度变化的速度和温度分布的不均匀程度成正比。
```

### 6.2 经济学

**边际分析：导数在经济决策中的核心作用**

```
成本函数 C(q) = 生产 q 个产品的总成本
边际成本 MC = dC/dq = 每多生产一个产品的额外成本

例子：
  C(q) = 1000 + 5q + 0.01q²
  MC = dC/dq = 5 + 0.02q

当 q=100 时，MC = 5 + 2 = 7 元/个
当 q=1000 时，MC = 5 + 20 = 25 元/个
→ 规模越大，边际成本越高
```

**利润最大化**

```
利润 π(q) = 收入 R(q) - 成本 C(q)

最优产量：dπ/dq = 0
即：边际收入 = 边际成本（MR = MC）

这是经济学中最核心的优化原则。
```

**复利与连续增长**

```
A(t) = P × e^(rt)

P = 本金
r = 年利率
t = 时间
A(t) = t 年后的总金额

导数 dA/dt = rA：钱的增长速度和你拥有的钱成正比。
```

### 6.3 工程学

**信号处理：从噪音中提取信号**

```
音频信号 s(t) = 原始声音 + 噪音

滤波器设计用微积分：
  - 低通滤波：保留低频，滤除高频（去掉尖锐噪音）
  - 高通滤波：保留高频，滤除低频（去掉嗡嗡声）

傅里叶变换（积分的一种形式）：
  S(ω) = ∫ s(t) × e^(-iωt) dt

把时域信号变成频域信号，看清每个频率成分的强度。
```

**控制系统：自动驾驶、机器人**

```
PID 控制器（最经典的控制算法）：

输出 = Kp × 误差 + Ki × ∫误差dt + Kd × d(误差)/dt
              ↑                    ↑
           积分项                  微分项

积分项：消除长期偏差（比如车一直偏右）
微分项：抑制振荡（让调整更平滑）
```

**结构工程：桥梁、建筑**

```
梁的弯曲方程：EI × d⁴y/dx⁴ = w(x)

E = 材料弹性模量
I = 截面惯性矩
y(x) = 梁在位置 x 的弯曲程度
w(x) = 分布载荷

工程师用这个方程计算：多大的力会让桥塌？
```

### 6.4 生物学与医学

**种群动力学：捕食者-猎物模型**

```
洛特卡-沃尔泰拉方程：

dx/dt = αx - βxy   （猎物的变化）
dy/dt = δxy - γy   （捕食者的变化）

x = 猎物数量，y = 捕食者数量
α = 猎物自然增长率
β = 捕食率
δ = 捕食者转化效率
γ = 捕食者死亡率

描述：捕食者多 → 猎物少 → 捕食者饿死 → 猎物恢复 → 捕食者增多...
      形成周期性振荡
```

**药物代谢：血药浓度**

```
C(t) = C₀ × e^(-kt)

C(t) = t 时刻血液中的药物浓度
C₀ = 初始浓度
k = 代谢速率常数

半衰期 t½ = ln(2)/k：药物浓度减半的时间

医生用这个决定：多久吃一次药？
```

**流行病学：SIR 模型**

```
dS/dt = -βSI       （易感者减少）
dI/dt = βSI - γI   （感染者变化）
dR/dt = γI         （康复者增加）

S = 易感人数，I = 感染人数，R = 康复人数
β = 传染率，γ = 康复率

基本传染数 R₀ = β/γ：一个感染者平均传染多少人

疫情预测的核心工具。
```

### 6.5 计算机图形学

**光线追踪：渲染真实图像**

```
计算光线和物体的交点：

光线方程：P(t) = O + tD    （起点 + 方向 × 距离）
球面方程：(P - C)·(P - C) = r²  （到球心距离 = 半径）

联立求 t：二次方程的解 = 光线和球的交点

需要求导找到法向量，计算反射/折射方向。
```

**动画：贝塞尔曲线**

```
贝塞尔曲线 B(t) = Σ C(n,i) × (1-t)^(n-i) × t^i × P_i

P_i = 控制点
t = [0, 1] 的参数

t = 0 时在起点，t = 1 时在终点
导数 B'(t) = 曲线的切线方向（速度方向）
二阶导数 B''(t) = 曲线的加速度方向

用来画平滑的曲线和动画路径。
```

**物理模拟：布料、流体、刚体**

```
布料模拟：每个顶点受重力、弹力、风力
  F = ma = m × d²x/dt²

流体模拟：纳维-斯托克斯方程
  ∂u/∂t + (u·∇)u = -∇p/ρ + ν∇²u + f

游戏里的布料飘动、水流、烟雾都是解这些微分方程。
```

### 6.6 金融学

**期权定价：Black-Scholes 模型**

```
欧式期权价格：

C = S₀N(d₁) - Ke^(-rT)N(d₂)

其中 d₁ = [ln(S₀/K) + (r + σ²/2)T] / (σ√T)
      d₂ = d₁ - σ√T

S₀ = 当前股价，K = 行权价
r = 无风险利率，T = 到期时间
σ = 股价波动率
N = 标准正态分布的累积分布函数（积分！）

导数（希腊字母）：
  Delta = ∂C/∂S   （股价变化1元，期权价格变化多少）
  Gamma = ∂²C/∂S² （Delta 的变化率）
  Theta = ∂C/∂t   （时间衰减）
  Vega = ∂C/∂σ    （波动率变化的影响）
```

**风险管理：VaR（风险价值）**

```
在 95% 的置信水平下，未来一天最多亏损多少钱？

VaR = μ + σ × Φ^(-1)(0.95)

μ = 平均收益，σ = 标准差
Φ^(-1) = 正态分布的反函数

需要计算投资组合价值的概率密度（积分）。
```

### 6.7 地理与气象

**天气预报：流体动力学方程**

```
大气运动用纳维-斯托克斯方程描述：
  ∂v/∂t + (v·∇)v = -∇p/ρ + g + 科里奥利力

v = 风速场，p = 气压，ρ = 密度，g = 重力

这是世界上最复杂的偏微分方程组之一。
天气预报就是解这个方程。
```

**海平面变化**

```
dh/dt = 冰川融化速率 + 海水热膨胀 + 地壳运动

需要积分计算长期累积效应。
```

### 6.8 统计学与概率

**概率密度函数（PDF）**

```
连续随机变量 X 的概率：
P(a ≤ X ≤ b) = ∫_a^b f(x) dx

f(x) = 概率密度函数
积分 = 求曲线下面积 = 概率

例子：正态分布
  f(x) = (1/√(2πσ²)) × e^(-(x-μ)²/(2σ²))
  
  P(X ≤ a) = ∫_(-∞)^a f(x) dx
```

**期望与方差**

```
期望 E[X] = ∫ x × f(x) dx    （平均值）
方差 Var(X) = ∫ (x - E[X])² × f(x) dx  （波动程度）

都是积分！
```

**贝叶斯推断**

```
P(假设|数据) = P(数据|假设) × P(假设) / P(数据)
                    ↑
                似然函数（概率密度）

后验分布 = 似然 × 先验 / 证据

证据 P(数据) = ∫ P(数据|假设) × P(假设) d(假设)  ← 积分！
```

## 七、微积分的核心思想总结

| 微积分操作 | 含义 | 应用领域 |
|-----------|------|---------|
| **导数** | 变化率 | 速度、边际成本、梯度下降 |
| **积分** | 累积 | 面积、概率、总量、工作量 |
| **微分方程** | 变化规律的描述 | 物理定律、生物模型、经济预测 |
| **优化** | 找最大值/最小值 | 利润最大化、损失最小化、路径最短 |

**一句话：微积分是研究"变化"的数学。只要一个量在变，微积分就能描述它。**
